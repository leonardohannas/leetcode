# LeetCode

This repository contains my personal solutions to exercises and problems inspired by the book *Beyond Cracking the Coding Interview*.

It serves as a space to document my practice, reasoning, and progress in algorithms, data structures, and technical interview preparation.

## Repository structure

Each folder represents one chapter or topic from the book.

Inside each chapter folder, I usually include:
- one `README.md` describing the chapter contents
- one `.py` file per exercise
- notes on approach and complexity
- runnable examples and test cases

## Current chapters

- `dynamic_arrays/`
- `string_manipulation/`

## Conventions

- One file per problem
- Problem summaries written in my own words
- Python implementations with complexity notes
- Test cases under `if __name__ == "__main__":`

## Goal

The goal of this repository is not only to solve problems, but also to better understand the underlying data structures, algorithmic trade-offs, and implementation details behind each solution.
