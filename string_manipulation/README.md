# String Manipulation

This folder contains my personal solutions for the **String Manipulation** chapter from *Beyond Cracking the Coding Interview*.

## Problems

- `string_manipulation.py` — introductory string manipulation exercises involving ASCII checks, character conversion, string splitting, joining, and substring search

## Current topics

This chapter currently includes practice with:

- checking whether a character is alphanumeric
- identifying character categories through ASCII ranges
- converting lowercase letters to uppercase
- solving character manipulation problems using only `ord()` and `chr()`
- manually splitting a string using a separator character
- manually joining an array of strings with a separator
- finding the first occurrence of a substring inside another string

## Notes

For each problem file, I keep:

- a short summary of the statement in my own words
- the Python implementation
- time and space complexity
- a runnable test section under `if __name__ == "__main__":`

## Goal

The goal of this folder is to strengthen my understanding of low-level string and character operations, including ASCII-based manipulation, manual string processing, and explicit substring search without relying on higher-level built-in methods.
