# String Manipulation

This folder contains my personal solutions for the **String Manipulation** chapter from *Beyond Cracking the Coding Interview*.

## Problems

- `01_character_manipulation_problem_set.py` — basic character manipulation exercises involving ASCII checks and character conversion

## Current topics

This chapter currently includes practice with:

- checking whether a character is alphanumeric
- identifying character categories through ASCII ranges
- converting lowercase letters to uppercase
- solving character manipulation problems using only `ord()` and `chr()`

## Notes

For each problem file, I keep:

- a short summary of the statement in my own words
- the Python implementation
- time and space complexity
- a runnable test section when appropriate

## Goal

The goal of this folder is to strengthen my understanding of low-level string and character operations, especially how characters can be manipulated through their numeric representations.
