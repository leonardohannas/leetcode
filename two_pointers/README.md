# Two Pointers

This folder contains my personal solutions for the **Two Pointers** chapter from *Beyond Cracking the Coding Interview*.

## Problems

- `two_pointers.py` — introductory two-pointer problems involving string and array traversal with coordinated indices

## Current topics

This chapter currently includes practice with:

- understanding the two-pointers technique
- using inward pointers
- checking whether a string is a palindrome
- comparing elements from both ends of a sequence
- solving problems with constant extra space

## Notes

For each problem file, I keep:

- a short summary of the statement in my own words
- the Python implementation
- time and space complexity
- a runnable test section under `if __name__ == "__main__":`

## Goal

The goal of this folder is to strengthen my understanding of the two-pointers technique and learn how different pointer movement patterns can simplify array and string problems while keeping solutions efficient and easy to reason about.
